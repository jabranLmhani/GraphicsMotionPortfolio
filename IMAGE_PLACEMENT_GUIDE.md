## üé® Image Quality Guidelines

### For Profile Photo:
1. **Resolution**: Minimum 1200x1600px (higher is better)
2. **File Size**: Under 500KB (optimize for web)
3. **Format**: JPG (for photos) or PNG (if transparency needed)
4. **Compression**: Use tools like TinyPNG or ImageOptim

### For Project Images:
1. **Resolution**: 1920x1080px or 2560x1440px
2. **File Size**: Under 300KB each
3. **Format**: JPG preferred
4. **Quality**: 80-90% JPEG quality
